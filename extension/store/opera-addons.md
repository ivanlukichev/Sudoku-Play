# Opera Add-ons Draft

## Short Description
Open Sudoku Play, Daily Sudoku, and Kids Sudoku directly from your Opera toolbar.

## Promo Text
Daily Sudoku one click away.

## Full Description
Sudoku Play for Opera gives you a fast launcher for the Sudoku Play website right from the browser toolbar.

Open the popup and choose what you want to play:

- Play Sudoku
- Daily Sudoku
- Kids Sudoku
- Guide

The extension is designed as a lightweight companion to https://sudoku-play.org/ so you can keep a quick logic break close by while browsing.

Highlights:

- compact popup UI
- one-click access to the main Sudoku board
- quick launch for the daily puzzle
- easy access to kids-friendly Sudoku modes
- link back to the full guide and website
