# Opera Add-ons Draft

## Short Description
Play a compact 9x9 Sudoku in Opera and open more puzzles on sudoku-play.org.

## Promo Text
Playable Sudoku one click away.

## Full Description
Sudoku Play for Opera gives you a compact playable Sudoku right from the browser toolbar.

Open the popup and choose what you want to play:

- Play the built-in Sudoku board
- Open Daily Sudoku on the website
- Open Kids Sudoku on the website
- Open the Guide

The extension is designed as a lightweight companion to https://sudoku-play.org/ so you can keep a quick logic break close by while browsing, then move to the website for more puzzle modes.

Highlights:

- compact popup UI
- built-in 9x9 Sudoku gameplay
- 16 preloaded puzzles
- number pad input and keyboard support
- quick launch for the daily puzzle
- easy access to kids-friendly Sudoku modes
- link back to the full guide and website
